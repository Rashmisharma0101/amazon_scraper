# Amazon Web Scraper
Overview
This project is a Python-based web scraper that extracts product information from Amazon and saves it into a CSV file for analysis. It is useful for tracking product prices, availability, and monitoring changes over time.

Features
- Extracts product title
- Extracts product price
- Saves data into CSV file
- Can be automated to run periodically
- Useful for price tracking and data analysis projects

Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- CSV

Use Cases
- Price tracking
- Market research
- Data analysis projects
- Learning web scraping
  
Future Improvements
- Scrape multiple products automatically
- Add email alerts for price drops
- Schedule automated scraping
- Store data in database
